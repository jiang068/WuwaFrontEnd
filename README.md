### WuwaFrontEnd 前端模板
鸣潮 bot 主人的大礼包~

---
#### 📖 使用方法
1. 鸣潮 bot 简介模板
直接对项目根目录进行修改定制即可，纯静态实现。

2. XutheringWavesUID 登录页模板
    `Example\XwUID_login_example`
直接放到 Gscore 控制台指示的登录页位置即可。

    `路径： gscore/data/XutheringwavesUID/show 目录`

3. 云游戏获取抽卡链接模板页
    `Example\Gacha_login_example`
给云游登录的作者准备的前端！但是我没给逆向过程和模拟登录的源码，无法索敌~ 只是给了你一个样式~
